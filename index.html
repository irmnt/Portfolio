HANDS-ON PROJECTS

ASP.NET - Car Loan Calculator

A hands-on project to improve my ASP.NET and C# skills with a loan calculator example. This application allows users to calculate their monthly car loan payments based on various factors such as loan amount, interest rate, tax rate, and loan term. It supports three different loan plans (Plan A, Plan B, Plan C), providing flexibility for users to compare payment options.

Java - Othello Game

A simple console-based Othello game where two players can compete against each other. The game includes valid move checks, automatic piece flipping, and game-over detection.

Windows Server Infrastructure Deployment Project

Configured a Windows Server core environment for a simulated enterprise (contoso.local), including Active Directory Domain Services and Group Policy to enforce password policies. Deployed and managed DHCP with multiple scopes, superscopes, and multicast scope; configured DNS forwarding; and created IP reservations and custom DHCP options. Set up a storage pool with a virtual disk, provisioned and attached iSCSI storage across servers, and implemented Hyper-V with VM creation and replication. Installed and configured Windows Deployment Services (WDS), applying Group Policy settings to enforce deployment rules and automate client behavior.

Microsoft 365 Environment Implementation Project

Designed and implemented a full Microsoft 365 environment for a fictional mid-sized organization (TechSolutions Inc., 300 users) to improve collaboration, security, and operational efficiency. Used the Microsoft 365 Admin Center and PowerShell to bulk import users, assign licenses, and configure Office 365 Groups and departmental permissions. Set up SharePoint sites, OneDrive policies, and Viva Engage for enterprise communication. Implemented Microsoft Defender for Office 365 with phishing/malware protections, configured message encryption, audit logs, DLP alerts, and usage reporting. Delivered complete system documentation and a stakeholder presentation demonstrating security compliance and service health monitoring.
